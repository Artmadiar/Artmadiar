# Obsidian Vault

Это Obsidian vault. Открой эту папку в Obsidian:

1. Открой Obsidian
2. "Open folder as vault"
3. Выбери папку `vault/`

## Рекомендуемые плагины

- **Calendar** — календарь в боковой панели
- **Periodic Notes** — автоматические daily/weekly notes
- **Dataview** — запросы по заметкам (для статистики привычек)
- **Obsidian Git** — автокоммит vault
- **Google Calendar** — интеграция с Google Calendar
- **Templater** — шаблоны для заметок

## Структура

- `daily/` — ежедневные заметки
- `projects/` — заметки по проектам
- `habits/` — трекинг привычек
